# 🛒 Google Boutique Microservices on AWS EKS

## 🚀 Project Overview
This repository contains the complete source code, Infrastructure as Code (IaC), and GitOps manifests for deploying a microservices-based e-commerce application on AWS.
*Note: In the actual deployment and video demo, I provisioned a self-managed GitLab server on an EC2 instance to implement the GitOps workflow and CI/CD pipelines.*

**Key Technologies:**
* **Cloud & Infrastructure:** AWS EKS, EC2, VPC, EBS, ElastiCache (Redis), Terraform
* **CI/CD & GitOps:** Self-managed GitLab CI, ArgoCD
* **Monitoring & Logging:** Grafana, EFK Stack (Elasticsearch, File Beat, Kibana)
